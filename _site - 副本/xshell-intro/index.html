具体操作：工具 – 选项 – 键盘和鼠标 – “按键对应”部分选择“编辑”按钮。

常用快捷键

快捷键 说明
Ctrl + inset 复制
Shift + insert 粘贴
Alt + O 新建（标签页）
ctrl + alt + N 新建窗口
Alt + left/right 两个xshell窗口之间进行切换
Ctrl + Tab 同一窗口中，不同标签之间的切换
Alt + Enter 全屏和取消全屏
