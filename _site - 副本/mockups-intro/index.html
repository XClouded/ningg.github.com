mockups的特点

  • 快：提供常用的元素，通过简单托、拉、拽等动作，即可完成线框图绘制；
  • 手绘风格：展示效果，朴素、简洁，能够突出内容、层次结构、业务流程；

高效用法

操作 说明
//+ 快速定位到Quick Add输入框，在其中输入控件名字，即可自动添加一个元素
Enter/F2 编辑
Ctrl+2 锁定
Ctrl+3 解除所有锁定
Ctrl+Up/Ctrl+Down 上移、下移一层
Ctrl+D 复制并粘贴，相当于Ctrl+C + Ctrl+V
Ctrl+G 多元素组合
Ctrl+Shift+G 取消组合
Ctrl+F 全屏显示，Esc取消全屏
Ctrl+L 显示\隐藏 UI Library
Ctrl+K 显示\隐藏 markup（标记）
Ctrl+1 Actual size
Ctrl+0 Zoom to fit
Ctrl+'+'/'-' Zoom In/Out
Ctrl+R Export to PNG，可以导出局部元素

（快捷操作，可以直接从菜单栏中查看，可以逐个尝试。）

  • 官方提供一些现有的UI模版：Mockups To Go
